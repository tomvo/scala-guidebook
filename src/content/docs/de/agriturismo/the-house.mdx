---
title: Das Haus
description: Trinkwasser und Zugang zum Erdgeschossbereich.
---

**Trinkwasser.** Das Leitungswasser ist gefiltert und enthärtet — Sie können es direkt aus dem Hahn trinken. Im Erdgeschossbereich finden Sie außerdem stilles und sprudelndes Wasser vom Hahn.

**Erdgeschossbereich.** Geöffnet von 07:30 bis 22:00. Falls die Tür während dieser Zeiten verschlossen ist, benutzen Sie das Tastenfeld: Geben Sie Ihren persönlichen Code ein und drücken Sie Yale zum Entriegeln.
